Implemented src/tokenize.rs: hand-written byte-cursor scanner that emits Token variants mirroring tokenize.sno names (ATOM/VAR/INT/NECK/LPAREN/RPAREN/COMMA/DOT plus LBRACKET/RBRACKET/PIPE/CUT/NOT/QUERY/EOF). Handles line and block comments, negative ints, and returns a BoundedArr<Token, 512>. 12 unit tests covering each token kind, full ancestor.pl with pinned count=58, and error cases (unterminated block comment, invalid char, overflow, ident-too-long).
